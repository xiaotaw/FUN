## FUN Project
A web server for **selfie2anime** based on [taki0112/UGATIT](https://github.com/taki0112/UGATIT/tree/e8efff198e252df0f3a5c936f02e7e7669264b13)

### 工作流程

用户上传图片 -> 本地保存 -> 面部识别 -> 转换成卡通风格 -> 本地保存 -> 返回网页显示
